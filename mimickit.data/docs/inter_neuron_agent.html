Inter Neuron Agent

Description: Interneuron agents are responsible for controlling connectivities between interneurons. They all are responsible for performing activity-independent (self-organizing) mechanisms of neural development to build an ab nihilo topology. An apt learning algorithm can also be applied to these agents to make the network evolve to a small-world topology.

State variables
Name Data type Initial value Value range Measure Description
potential int -70 [-70..30] mV Potential indicates the membrane potential of this neuron. It changes between -70 mV and 30 mV. A neuron has normally (in resting state) got a potential of -70. When a neuron is excited its membrane potential goes up to 30 and goes down to -70 again (neuronal signal).

Nominal Behaviors
Action Name Input description Precondition Intended effect
Conduct spikes - Stimulation by a neuron. Conducts incoming spike to the neurons in which it has synaptic connection.

Evolution Behaviors
Action Name Input description Precondition Intended effect
Create a New Interneuron - Detection of the "Lack of Pathway" NCS. Since its axonal delay is fixed, it creates a new interneuron.


Interface
