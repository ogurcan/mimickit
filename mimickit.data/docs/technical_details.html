Technical Details

Limitiations of RePast Simulation Framework

Since RePast 1.2.0 does not allow scheduled actions to be defined in a parent class, the implementation of some generic methods (such as controlInLinks() of Neuron class) are done at each of its subclasses. This limitation prevents code re-use, thus reduce the flexiblity and maintainability of the simulation model.
